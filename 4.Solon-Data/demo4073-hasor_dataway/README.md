# 说明
> 本项目是基于Solon框架实现hasor-dataway的demo，主要是为了演示hasor-dataway的使用。方便快速上手Solon框架的学习

# 使用

## 1、配置表的新建
将 `resouce/db/schema-myqsql.sql` 导入元数据库

## 2、配置多数据源
在 `resouce/app.yml` 中配置多数据源

## 3、启动服务
启动服务后，打开：
http://localhost:8080/interface-ui/

账号：admin

密码：admin
